# Money Manager
---

## Key Features

* **Transaction Management:** Seamlessly log income and expenses with category tagging.
* **Data Visualization:** Dynamic dashboards using **Chart.js** and **PrimeNG Charts** to track monthly trends.
* **Secure Authentication:** Protected by **Spring Security** and **JWT**.
* **Financial Reports:** Export your data to Excel (.xlsx) using **Apache POI**.
* **Email Integration:** Automated notifications via **Spring Mail**.
* **Modern UI/UX:** Built with **PrimeNG**, **Tailwind CSS**, and **Lucide Icons** for a premium feel.
* **Global Support:** Multi-language capabilities allowed by **ngx-translate**.

---

## Tech Stack

### Frontend
- **Framework:** Angular 21
- **Styling:** Tailwind CSS
- **UI Components:** PrimeNG & PrimeUI Themes

### Backend
- **Framework:** Spring Boot
- **Security:** Spring Security, JWT
- **Database:** MySQL (testing) / PostgreSQL (prod) (JPA & Hibernate)
- **Document Processing:** Apache POI (Excel)
- **Communication:** Spring Boot Starter Mail
